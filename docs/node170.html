Input

COMBINE is called by the command line 

combine <infile>

where <infile> is the name (or path, if desired) of a SIGSPEC result file or an output file generated by CINDERELLA.



Caution: COMBINE overwrites existing output files!

Furthermore, the user may pass a set of specifications to COMBINE by means of a file <infile>.ini in the same folder as <infile>. For specifications not given by the user, defaults are used.



The file <infile>.ini has to be terminated by a carriage-return character, otherwise the program hangs!
