profile

SIGSPEC generates a file profile.dat containing the sampling profile for the given time series, p.. By default, the file profile.dat is not generated.

This keyword is ignored in MultiFile mode, where sampling profiles are calculated and written to files whenever required by the program. See ``MultiFile Mode'', p. for further information.
