results <int> <int> ()

output files containing a list of significant signal components. The default setting is to produce only a file result.dat for the final list. The parameters are

  1. the maximum number of iterations for which to write additional result files r#iteration#.dat, and
  2. the step width (number of iterations) for output, p..

The file name may be assigned additional indices for Time-Resolved analysis (p.) and/or MultiFile mode (p.).
