Time series columns representing time and observable

The keywords col:time and col:obs determine those columns in the time series input file which contain time values and the observable monitored over time, respectively. These columns are required and have to be uniquely specified. Column indices start with 1.

If col:time is not specified, the default value is used. If col:obs is not specified, the default value is used.



Example. The sample project coltimecolobs contains a dataset where the time and observable values are found in columns 2 and 3, respectively. The input time series represents the V photometry of IC4996#89 (see Example SigSpecNative, p.). The file coltimecolobs.ini contains the two lines 

col:time 2
col:obs 3
