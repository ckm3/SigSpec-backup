Comparison datasets

The keyword comp in the .cnd file is used for the specification of a comparison dataset. The keyword is followed by an integer value referring to the six-digit index of the desired time series input file. Contrary to the file nomenclature, the six-digit format is not required for file indices specified in the .cnd file. If no .cnd file is available, CINDERELLA uses all time series input files - except for the first one, which is considered target data - as comparison datasets.
