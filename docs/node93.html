residuals <int> <int> ()

output files containing residual time series (only residuals.dat for the residuals after prewhitening all significant compontents by default). The parameters are

  1. the maximum number of iterations (files t#iteration#.dat), and
  2. the step width (number of iterations) for output, p..

The file name may be assigned additional indices for Time-Resolved analysis (p.) and/or MultiFile mode (p.).
