The time series input file

The input file for SIGSPEC is a time series. The corresponding file has to be located in the same folder as the project directory. The only restrictions to the format are that the number of items per row has to be constant for all rows in the file and that columns have to be separated by at least one whitespace character or tab. Dataset entries need not be numeric, except for the columns specified as time, observable, and weights (p.).
