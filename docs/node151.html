Keywords Reference

This section is a compilation of all keywords accepted by CINDERELLA. A brief description of arguments and default values is given. The type of argument is provided by either <int>, <double>, or <string>, and default values are given in parentheses, e.g. (2). Empty parentheses indicate that there is no default setting.
