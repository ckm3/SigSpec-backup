mstracks <int> <int> ()

MultiSine tracks are written to files m#index#.dat, where #index# refers to the signal component in the result files. The parameters are

  1. the maximum number of iterations for which to write entries to the MultiSine track files, and
  2. the step width (number of iterations) for output, p..

The file name may be assigned additional indices for Time-Resolved analysis (p.) and/or MultiFile mode (p.).
