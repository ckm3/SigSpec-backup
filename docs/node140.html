Datasets to ignore

The keyword skip in the .cnd file is used for the specification of a dataset not to be taken into account for computation. The keyword is followed by an integer value referring to the six-digit index of the desired time series input file. Contrary to the file nomenclature, the six-digit format is not required for file indices specified in the .cnd file.
